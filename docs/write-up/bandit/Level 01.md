# Bandit Level 1


(важно снова авторизироваться при помощи пароля, который мы получили на прошлом уровне и команды)
```bash
ssh bandit[1]{.mark}@bandit.labs.overthewire.org -p 2220
```
_*и каждый следующий уровень нам надо менять выделенную цифру на соответствующий уровень_

На этом уровне нам говорят, что следующий пароль хранится в файле в домашнем каталоге, поэтому мы используем команду
```bash
cd ../
```
две точки позволяют нам подняться на каталог выше, поэтому мы попадаем в home, хотя также можно было использовать
```bash
cd /home
```
Попав в домашний каталог, мы видим большое количество файлов (рис 1.1)

![Рисунок 1.1](media/level_0-33/media/image3.png)

Откроем каталог **bandit1** при помощи команды
```bash
cd /home/bandit1
```
и при помощи команды **ls** найдем в нем файл **--**

и при помощи команды
```bash
cat ./-
```
откроем этот файл и получим пароль

**Ответ: `263JGJPfgU6LtdEvgfWU1XP5yac29mFx`**
